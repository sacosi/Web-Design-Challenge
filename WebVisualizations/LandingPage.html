Summary: Latitude vs. X


The purpose of this project was to analyze how weather changes as you get closer or futher way from the equator.

To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Python and the Matplotlib library to plot various aspects of the weather vs. latitude. Factors we looked at included: Temperature, Humidity, Cloudiness and Wind Speed.

This site provides the source data and visualizations created as part of the anlysis, as well as explanations and descriptions of any trends and correlations witnessed.

By clicking on any of the plots in the visualizations box or by going under the menu plots, you can find a more detailed analysis on each of the 4 factors that were analyzed and mentioned above.

The Comparisson menu will show you the 4 graphs on one page for a better visualization on how the latitude affects each of the 4 factors. Just click on any of the images, should you want to do a deeper dive.

You will find the full dataset under the Data menu. For every city we have the factors mentioned above as well as its country, latitude, longitude and the date when these observations were taken.
